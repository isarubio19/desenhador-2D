Desenvolvimento de sistema gráfico interativo em Java com Swing, manipulação de eventos de mouse e programação orientada a objetos. Implementação de editor para criação de primitivas geométricas (pontos, retas, triângulos, círculos e retângulos), armazenamento das figuras em estruturas de dados, renderização dinâmica e funcionalidades de seleção, remoção e persistência de desenhos.
